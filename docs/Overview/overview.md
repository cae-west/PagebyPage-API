# Page by Page API overview

The Page by Page service is a REST API that transforms reading from a passive hobby into an active practice.

## Great for casual readers and avid readers

Start reading more or keep up your reading streak. Track the pages you finish and time you spend on each book. Write notes about your thoughts and feelings as you read. See how your reading habit grows over time.

## Motivate yourself to commit to a new reading habit

Give your mind a break and build meaningful reading habits. The Page by Page API helps reduce stress and anxiety related to [doomscrolling](https://www.psychologytoday.com/us/blog/its-not-just-in-your-head/202412/the-dangers-of-doom-scrolling?msockid=26e2d2a4ac14693428bbc722adf068f7) and [brain rot](https://letterpressproject.co.uk/media/file/The_Benefits_of_Reading_for_Pleasure.pdf).
